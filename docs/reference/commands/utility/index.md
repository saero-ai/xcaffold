# Utility Commands

Commands for inspecting schemas, exporting plugins, and running agent simulations.

| Command | Action |
| --- | --- |
| [`export`](/docs/cli/reference/commands/utility/export) | *(Preview)* Package compiled output as a distributable plugin. |
| [`help`](/docs/cli/reference/commands/utility/help) | Display general help or schema documentation for xcaffold resource kinds. |
| [`test`](/docs/cli/reference/commands/utility/test) | *(Preview)* Run a sandboxed local simulation of a Claude agent. |
