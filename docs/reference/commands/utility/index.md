# Utility Commands

Commands for schema inspection, project registry management, and documentation.

| Command | Action |
| --- | --- |
| [`help`](/docs/cli/reference/commands/utility/help) | Display general help or schema documentation for xcaffold resource kinds. |
| [`registry`](/docs/cli/reference/commands/utility/registry) | Manage the global project registry — list, add, remove, prune, and inspect projects. |
