# Security Policy

## Reporting a Vulnerability

We take the security of xcaffold seriously. If you discover a security vulnerability, please do NOT report it by opening a public GitHub issue.

Instead, please send an email to security@saero.ai. We will acknowledge your report within 48 hours and provide a timeline for a fix. Requesting that you do not publicly disclose the issue until we have released a patch.

## Supported Versions

Currently, we only provide security updates for the latest released version of the CLI.
